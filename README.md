# Restaurant Menu Assessment

## What to Build

A restaurant menu page using HTML and CSS.

## Requirements

### HTML
- A header with your restaurant name and a short slogan e.g. *"Home cooked, heart approved"*
- A Starters section with at least 3 items
- A Main Course section with at least 3 items
- A Desserts section with at least 3 items
- A Find Us section with a short description and a link

### CSS
- Each section in its own `div` with a class
- A border, padding, and margin on each section
- Flexbox on the header — restaurant name and slogan side by side
- A background colour on the page that is different from the sections
- At least one use of `border-radius`, `text-align`, and `line-height`

## Folder Structure

```
your-name-restaurant/
├── index.html
└── styles.css
```

## Submission

Fork this repo, build your page, then share your fork link.
